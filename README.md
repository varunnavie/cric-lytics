#  Cricket Match Prediction Dashboard

An end-to-end Machine Learning powered cricket match prediction system with AI explainability, built using Streamlit.

##  Features
- First innings score prediction
- Chasing team score & win probability
- Playing XI recommendation using clustering
- SHAP-based AI explainability
- Interactive Streamlit dashboard

##  Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (ML models)
- SHAP (Explainable AI)
- Streamlit (Dashboard)
